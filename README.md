# From The Heart (FTH)

This repository contains the product documentation and implementation blueprint for **From The Heart**, a premium gifting platform with personalized websites, QR-powered gift experiences, and physical gift combos.

## Repository Structure

- `docs/` – Business, product, and technical documentation
- `database/` – SQL schema and data model notes
- `api/` – API specification
- `diagrams/` – Architecture and user-flow diagrams

## Project Goals

- Sell physical gifts and gift hampers
- Offer personalized celebration websites
- Support gift + website combo orders
- Generate dynamic QR cards for combo orders
- Provide guest checkout with smart order lookup
- Include a configurable admin panel

## Tech Stack

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Hostinger-compatible deployment
